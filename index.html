1Sistemas Operativos y Redes

Sistema operativo es aquel software que permite unir las funciones y herramientas de los equipos, las aplicaciones y se comunica directamente con el dispositivo para saber si hay alguna inconsistencia en la comunicación.

2Diseño y Desarrollo Web

Es un área enfocada en el desarrollo de interfaces digitales, como el diseño de sitios y aplicaciones para web. Para ello, los diseñadores web crean las páginas utilizando lenguajes de marcado como HTML.

3Programación

Es el arte del proceso por el cual se limpia, codifica, traza y protege el código fuente de programas computacionales, en otras palabras, es indicarle a la computadora lo que tiene que hacer.
